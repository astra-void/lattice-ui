Lattice UI is a collection of headless primitives for building composable Roblox interfaces with `@rbxts/react`.

This docs site is CLI-first and designed for both app developers and monorepo contributors.

## What You Can Find Here

- Installation and quick-start flows using the `lattice` CLI.
- Command reference for `create`, `add`, `upgrade`, and `doctor`.
- Component and preset mapping sourced from the CLI registry.
- Contributor-only workspace governance and release guidance.
