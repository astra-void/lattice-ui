Lattice UI is a set of headless Roblox UI primitives built for composable React patterns.
The docs are organized around how teams actually work: scaffold, add, remove, validate, and upgrade.

## What You Can Find Here

- End-to-end setup with `create`, `add`, `remove`, `doctor`, and `upgrade`.
- Command-level behavior rules, including interactive prompts and `--yes` constraints.
- Registry-backed component and preset references used by the CLI.
- Component pages with install commands, primary exports, and usage snippets.
- Contributor-only monorepo governance and release guidance.
