## Global Options

Supported options:

- `--help`
- `--version`

## `create`

Create a new project from the rbxts starter template.

### Global CLI

```bash
lattice create my-game --pm npm
```

### Monorepo Local Script

```bash
pnpm lattice create my-game --pm npm
```

## `add`

Add components by name and/or preset.

### Global CLI

```bash
lattice add dialog,toast --preset overlay
```

### Monorepo Local Script

```bash
pnpm lattice add dialog,toast --preset overlay
```

## `upgrade`

Upgrade installed Lattice UI packages in the current project.

### Global CLI

```bash
lattice upgrade
lattice upgrade --preset form
```

### Monorepo Local Script

```bash
pnpm lattice upgrade
pnpm lattice upgrade --preset form
```

## `doctor`

Run diagnostics for lockfile/package manager consistency, missing peers, and provider expectations.

### Global CLI

```bash
lattice doctor
```

### Monorepo Local Script

```bash
pnpm lattice doctor
```
