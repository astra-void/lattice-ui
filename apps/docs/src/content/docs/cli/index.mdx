Audience: app developers integrating Lattice UI into project codebases.

The `lattice` CLI is the recommended way to scaffold and maintain Lattice UI dependencies.

## When to Use CLI

- Initialize a new project baseline (`init`)
- Install component dependencies by name or preset (`add`)
- Upgrade installed `@lattice-ui/*` packages (`upgrade`)
- Diagnose package manager and dependency issues (`doctor`)

## Execution Modes

### Global CLI

```bash
npm i -g @lattice-ui/cli
lattice --help
```

### Monorepo Local Script

```bash
pnpm lattice --help
```
