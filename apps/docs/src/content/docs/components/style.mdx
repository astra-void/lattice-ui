## Install

### Global CLI

```bash
lattice add style
```

### Monorepo Local Script

```bash
pnpm lattice add style
```

## Package

- npm: `@lattice-ui/style`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Usage Snippet

```tsx
import { Box, Text, ThemeProvider, createTheme } from "@lattice-ui/style";

const theme = createTheme();

export function StyledCard() {
  return (
    <ThemeProvider theme={theme}>
      <Box
        sx={(t) => ({
          BackgroundColor3: t.colors.surface,
          Size: UDim2.fromOffset(260, 96),
        })}
      >
        <Text
          sx={(t) => ({
            TextColor3: t.colors.text,
            TextSize: t.typography.body.fontSize,
          })}
        >
          Themed content
        </Text>
      </Box>
    </ThemeProvider>
  );
}
```


## Notes

- Use `ThemeProvider` plus token creators to drive consistent visual language.
- `createRecipe` and `sx` helpers reduce repeated style object composition.
