## Install

### Global CLI

```bash
lattice add toast
```

### Monorepo Local Script

```bash
pnpm lattice add toast
```

## Package

- npm: `@lattice-ui/toast`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/layer`

## Primary Exports

- `Toast`
- `Toast.Provider`
- `Toast.Viewport`
- `Toast.Root`
- `Toast.Title`
- `Toast.Description`
- `Toast.Action`
- `Toast.Close`
- `useToast`

## Usage Snippet

```tsx
import { PortalProvider } from "@lattice-ui/layer";
import { Toast } from "@lattice-ui/toast";

export function SaveToast({ playerGui }: { playerGui: PlayerGui }) {
  return (
    <PortalProvider container={playerGui}>
      <Toast.Provider>
        <Toast.Viewport />
        <Toast.Root>
          <Toast.Title asChild>
            <textlabel Text="Saved" />
          </Toast.Title>
          <Toast.Description asChild>
            <textlabel Text="Settings updated." />
          </Toast.Description>
        </Toast.Root>
      </Toast.Provider>
    </PortalProvider>
  );
}
```


## Notes

- Imperative API is provided through `useToast()`.
- Default toast timing is `durationMs=4000` with `maxVisible=3`.
- Queue helpers are exported for deterministic unit testing.
- Toast viewport can be mounted with PortalProvider for layering.
