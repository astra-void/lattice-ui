## Install

### Global CLI

```bash
lattice add progress
```

### Monorepo Local Script

```bash
pnpm lattice add progress
```

## Package

- npm: `@lattice-ui/progress`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Primary Exports

- `Progress`
- `Progress.Root`
- `Progress.Indicator`
- `Progress.Spinner`

## Usage Snippet

```tsx
import { Progress } from "@lattice-ui/progress";

export function UploadProgress() {
  return (
    <Progress.Root max={100} value={45}>
      <Progress.Indicator asChild>
        <frame Size={UDim2.fromScale(1, 1)} />
      </Progress.Indicator>
    </Progress.Root>
  );
}
```


## Notes

- `Progress` supports `value`/`max` and clamps out-of-range values.
- `indeterminate` mode uses a fixed indicator ratio.
- `Spinner` defaults to `spinning=true` and rotates on heartbeat.
