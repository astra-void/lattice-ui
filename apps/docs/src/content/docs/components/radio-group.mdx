## Install

### Global CLI

```bash
lattice add radio-group
```

### Monorepo Local Script

```bash
pnpm lattice add radio-group
```

## Package

- npm: `@lattice-ui/radio-group`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`

## Primary Exports

- `RadioGroup`
- `RadioGroup.Root`
- `RadioGroup.Item`
- `RadioGroup.Indicator`

## Usage Snippet

```tsx
import { RadioGroup } from "@lattice-ui/radio-group";

export function DifficultyRadio() {
  return (
    <RadioGroup.Root defaultValue="normal" orientation="horizontal">
      <RadioGroup.Item value="easy">
        <RadioGroup.Indicator asChild>
          <textlabel Text="●" />
        </RadioGroup.Indicator>
      </RadioGroup.Item>
      <RadioGroup.Item value="normal">
        <RadioGroup.Indicator asChild>
          <textlabel Text="●" />
        </RadioGroup.Indicator>
      </RadioGroup.Item>
    </RadioGroup.Root>
  );
}
```


## Notes

- `RadioGroup.Root` owns the selected value and registration of items.
- `RadioGroup.Indicator` allows custom visuals while preserving accessibility semantics.
