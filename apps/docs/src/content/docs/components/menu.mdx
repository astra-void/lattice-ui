## Install

### Global CLI

```bash
lattice add menu
```

### Monorepo Local Script

```bash
pnpm lattice add menu
```

## Package

- npm: `@lattice-ui/menu`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`, `@lattice-ui/layer`, `@lattice-ui/popper`

## Primary Exports

- `Menu`
- `Menu.Root`
- `Menu.Trigger`
- `Menu.Portal`
- `Menu.Content`
- `Menu.Item`
- `Menu.Group`
- `Menu.Label`
- `Menu.Separator`

## Notes

- Compose `Menu.Trigger`, `Menu.Content`, and `Menu.Item` for context menus or dropdown menus.
- Grouping and labeling primitives help structure complex menu trees.
- Menu.Content can use PortalProvider for display order control.
