## Install

### Global CLI

```bash
lattice add checkbox
```

### Monorepo Local Script

```bash
pnpm lattice add checkbox
```

## Package

- npm: `@lattice-ui/checkbox`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Primary Exports

- `Checkbox`
- `Checkbox.Root`
- `Checkbox.Indicator`

## Usage Snippet

```tsx
import { Checkbox } from "@lattice-ui/checkbox";

export function TermsCheckbox() {
  return (
    <Checkbox.Root defaultChecked={false}>
      <Checkbox.Indicator asChild>
        <textlabel Text="✓" />
      </Checkbox.Indicator>
    </Checkbox.Root>
  );
}
```

## Notes

- `Checkbox.Root` owns interaction and checked-state behavior.
- `Checkbox.Indicator` reads checkbox state from context so visuals stay in sync.
