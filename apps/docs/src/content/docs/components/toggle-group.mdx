## Install

### Global CLI

```bash
lattice add toggle-group
```

### Monorepo Local Script

```bash
pnpm lattice add toggle-group
```

## Package

- npm: `@lattice-ui/toggle-group`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`

## Primary Exports

- `ToggleGroup`
- `ToggleGroup.Root`
- `ToggleGroup.Item`

## Notes

- Supports grouped toggle interactions through shared root context.
- Use for segmented controls and multi-select button groups.
