## Install

### Global CLI

```bash
lattice add toggle-group
```

### Monorepo Local Script

```bash
pnpm lattice add toggle-group
```

## Package

- npm: `@lattice-ui/toggle-group`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`

## Primary Exports

- `ToggleGroup`
- `ToggleGroup.Root`
- `ToggleGroup.Item`

## Usage Snippet

```tsx
import { ToggleGroup } from "@lattice-ui/toggle-group";

export function TextStyleToggle() {
  return (
    <ToggleGroup.Root type="multiple" defaultValue={["bold"]}>
      <ToggleGroup.Item asChild value="bold">
        <textbutton Text="B" />
      </ToggleGroup.Item>
      <ToggleGroup.Item asChild value="italic">
        <textbutton Text="I" />
      </ToggleGroup.Item>
      <ToggleGroup.Item asChild value="underline">
        <textbutton Text="U" />
      </ToggleGroup.Item>
    </ToggleGroup.Root>
  );
}
```

## Notes

- Supports grouped toggle interactions through shared root context.
- Use for segmented controls and multi-select button groups.
