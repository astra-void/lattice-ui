## Install

### Global CLI

```bash
lattice add layer
```

### Monorepo Local Script

```bash
pnpm lattice add layer
```

## Package

- npm: `@lattice-ui/layer`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Primary Exports

- `DismissableLayer`
- `Portal`
- `PortalProvider`
- `Presence`

## Usage Snippet

```tsx
import { DismissableLayer, Portal, PortalProvider, Presence } from "@lattice-ui/layer";

export function LayeredPanel({ playerGui }: { playerGui: PlayerGui }) {
  return (
    <PortalProvider container={playerGui}>
      <Presence present>
        <Portal>
          <DismissableLayer>
            <textlabel Text="Overlay content" />
          </DismissableLayer>
        </Portal>
      </Presence>
    </PortalProvider>
  );
}
```

## Notes

- `PortalProvider` helps centralize overlay mounting and display-order management.
- `DismissableLayer` and `Presence` are shared by overlay components such as dialog/popover.
