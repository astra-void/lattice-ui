## Install

### Global CLI

```bash
lattice add accordion
```

### Monorepo Local Script

```bash
pnpm lattice add accordion
```

## Package

- npm: `@lattice-ui/accordion`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`

## Primary Exports

- `Accordion`
- `Accordion.Root`
- `Accordion.Item`
- `Accordion.Header`
- `Accordion.Trigger`
- `Accordion.Content`

## Usage Snippet

```tsx
import { Accordion } from "@lattice-ui/accordion";

export function SettingsAccordion() {
  return (
    <Accordion.Root type="single" defaultValue="audio">
      <Accordion.Item value="audio">
        <Accordion.Header>
          <Accordion.Trigger asChild>
            <textbutton Text="Audio" />
          </Accordion.Trigger>
        </Accordion.Header>
        <Accordion.Content asChild>
          <textlabel Text="Audio settings" />
        </Accordion.Content>
      </Accordion.Item>
    </Accordion.Root>
  );
}
```


## Notes

- Supports `type="single" | "multiple"`.
- Supports controlled/uncontrolled `value`.
- Keyboard navigation is integrated with roving focus.
