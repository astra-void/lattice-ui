## Install

### Global CLI

```bash
lattice add focus
```

### Monorepo Local Script

```bash
pnpm lattice add focus
```

## Package

- npm: `@lattice-ui/focus`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Primary Exports

- `FocusScope`
- `RovingFocusGroup`
- `RovingFocusItem`

## Usage Snippet

```tsx
import { FocusScope, RovingFocusGroup, RovingFocusItem } from "@lattice-ui/focus";

export function FocusToolbar() {
  return (
    <FocusScope trapped restoreFocus>
      <RovingFocusGroup orientation="horizontal" loop>
        <RovingFocusItem asChild>
          <textbutton Text="Left" />
        </RovingFocusItem>
        <RovingFocusItem asChild>
          <textbutton Text="Center" />
        </RovingFocusItem>
        <RovingFocusItem asChild>
          <textbutton Text="Right" />
        </RovingFocusItem>
      </RovingFocusGroup>
    </FocusScope>
  );
}
```


## Notes

- `RovingFocusGroup` and `RovingFocusItem` provide arrow/Home/End keyboard navigation.
- `FocusScope` can trap and restore `GuiService.SelectedObject` across scope lifecycle.
