## Install

### Global CLI

```bash
lattice add dialog
```

### Monorepo Local Script

```bash
pnpm lattice add dialog
```

## Package

- npm: `@lattice-ui/dialog`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`, `@lattice-ui/layer`

## Primary Exports

- `Dialog`
- `Dialog.Root`
- `Dialog.Trigger`
- `Dialog.Portal`
- `Dialog.Content`
- `Dialog.Overlay`
- `Dialog.Close`

## Usage Snippet

```tsx
import { Dialog } from "@lattice-ui/dialog";
import { PortalProvider } from "@lattice-ui/layer";

export function ConfirmDialog({ playerGui }: { playerGui: PlayerGui }) {
  return (
    <PortalProvider container={playerGui}>
      <Dialog.Root defaultOpen={false}>
        <Dialog.Trigger asChild>
          <textbutton Text="Open dialog" />
        </Dialog.Trigger>
        <Dialog.Portal>
          <Dialog.Overlay />
          <Dialog.Content>
            <textlabel Text="Are you sure?" />
            <Dialog.Close asChild>
              <textbutton Text="Close" />
            </Dialog.Close>
          </Dialog.Content>
        </Dialog.Portal>
      </Dialog.Root>
    </PortalProvider>
  );
}
```


## Notes

- `Dialog.Root` supports controlled and uncontrolled open state.
- `Dialog.Content` and `Dialog.Overlay` are designed to compose with layer primitives.
- Dialog.Content can use PortalProvider for predictable layering.
