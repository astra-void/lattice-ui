## Install

### Global CLI

```bash
lattice add slider
```

### Monorepo Local Script

```bash
pnpm lattice add slider
```

## Package

- npm: `@lattice-ui/slider`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/focus`

## Primary Exports

- `Slider`
- `Slider.Root`
- `Slider.Track`
- `Slider.Range`
- `Slider.Thumb`

## Usage Snippet

```tsx
import { Slider } from "@lattice-ui/slider";

export function VolumeSlider() {
  return (
    <Slider.Root min={0} max={100} value={35}>
      <Slider.Track asChild>
        <frame Size={UDim2.fromOffset(220, 12)}>
          <Slider.Range asChild>
            <frame />
          </Slider.Range>
          <Slider.Thumb asChild>
            <textbutton Text="" />
          </Slider.Thumb>
        </frame>
      </Slider.Track>
    </Slider.Root>
  );
}
```

## Notes

- Single-thumb only in this release.
- Supports controlled/uncontrolled `value`.
- Pointer drag and keyboard adjustments are both supported.
- Values are clamped to `[min, max]` and snapped to `step`.
