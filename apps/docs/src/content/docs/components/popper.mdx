## Install

```bash
lattice add popper
```

```bash
pnpm lattice add popper
```

## Package

- npm: `@lattice-ui/popper`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Primary Exports

- `usePopper`
- `compute placement utilities`
- `Popper types`

## Usage Snippet

```tsx
import { React } from "@lattice-ui/core";
import { usePopper } from "@lattice-ui/popper";

export function AnchoredOverlay() {
  const anchorRef = React.useRef<GuiObject>();
  const contentRef = React.useRef<GuiObject>();
  const { position, anchorPoint } = usePopper({
    anchorRef,
    contentRef,
    placement: "bottom",
    offset: new Vector2(0, 8),
  });

  return (
    <frame>
      <textbutton ref={anchorRef} Text="Anchor" />
      <frame ref={contentRef} Position={position} AnchorPoint={anchorPoint}>
        <textlabel Text="Positioned with usePopper" />
      </frame>
    </frame>
  );
}
```


## Notes

- Provides the positioning core used by anchored overlays like Select, Tooltip, and Popover.
- Pair with `@lattice-ui/layer` portals for predictable overlay placement.
