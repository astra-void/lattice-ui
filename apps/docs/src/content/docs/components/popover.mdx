## Install

### Global CLI

```bash
lattice add popover
```

### Monorepo Local Script

```bash
pnpm lattice add popover
```

## Package

- npm: `@lattice-ui/popover`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`, `@lattice-ui/layer`, `@lattice-ui/popper`

## Primary Exports

- `Popover`
- `Popover.Root`
- `Popover.Trigger`
- `Popover.Portal`
- `Popover.Content`
- `Popover.Anchor`
- `Popover.Close`

## Usage Snippet

```tsx
import { Popover } from "@lattice-ui/popover";
import { PortalProvider } from "@lattice-ui/layer";

export function HelpPopover({ playerGui }: { playerGui: PlayerGui }) {
  return (
    <PortalProvider container={playerGui}>
      <Popover.Root>
        <Popover.Trigger asChild>
          <textbutton Text="Show help" />
        </Popover.Trigger>
        <Popover.Portal>
          <Popover.Content asChild>
            <textlabel Text="Popover content" />
          </Popover.Content>
        </Popover.Portal>
      </Popover.Root>
    </PortalProvider>
  );
}
```


## Notes

- Use `Popover.Anchor` when the positioning reference differs from the trigger element.
- `Popover.Close` gives an explicit close action from inside content.
- Popover.Content can safely manage displayOrder with PortalProvider.
