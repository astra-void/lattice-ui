## Install

### Global CLI

```bash
lattice add core
```

### Monorepo Local Script

```bash
pnpm lattice add core
```

## Package

- npm: `@lattice-ui/core`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `None`

## Primary Exports

- `createContextScope and context helpers`
- `Slot and Slottable primitives`
- `composeRefs and ref helpers`
- `useControllableState`
- `React and ReactRoblox bindings`

## Usage Snippet

```tsx
import { React, useControllableState } from "@lattice-ui/core";

export function ControllableExample() {
  const [value, setValue] = useControllableState({
    defaultValue: "alpha",
  });

  return (
    <textbutton
      Text={value}
      Event={{
        Activated: () => setValue((current) => (current === "alpha" ? "beta" : "alpha")),
      }}
    />
  );
}
```


## Notes

- This package is shared infrastructure used by all other component packages.
- Use it directly when building custom compound components or low-level primitives.
