## Install

### Global CLI

```bash
lattice add scroll-area
```

### Monorepo Local Script

```bash
pnpm lattice add scroll-area
```

## Package

- npm: `@lattice-ui/scroll-area`
- peer dependencies: `@rbxts/react`, `@rbxts/react-roblox`
- internal dependencies: `@lattice-ui/core`

## Primary Exports

- `ScrollArea`
- `ScrollArea.Root`
- `ScrollArea.Viewport`
- `ScrollArea.Scrollbar`
- `ScrollArea.Thumb`
- `ScrollArea.Corner`

## Usage Snippet

```tsx
import { ScrollArea } from "@lattice-ui/scroll-area";

export function InventoryList() {
  return (
    <ScrollArea.Root>
      <ScrollArea.Viewport asChild>
        <scrollingframe Size={UDim2.fromOffset(220, 140)}>
          <frame Size={UDim2.fromOffset(220, 420)} />
        </scrollingframe>
      </ScrollArea.Viewport>

      <ScrollArea.Scrollbar orientation="vertical">
        <ScrollArea.Thumb orientation="vertical" />
      </ScrollArea.Scrollbar>
    </ScrollArea.Root>
  );
}
```

## Notes

- Supports vertical and horizontal scrollbar primitives.
- `type` supports `auto`, `always`, and `scroll` visibility policies.
- Thumb/canvas math helpers are exported for unit testing.
