## Prerequisites

- Node.js 20+
- One package manager: `npm`, `pnpm`, or `yarn`
- A Roblox TS project using `@rbxts/react` and `@rbxts/react-roblox`

## Execution Mode Matrix

| Mode                  | Best for                               | First setup                | Command style            |
| --------------------- | -------------------------------------- | -------------------------- | ------------------------ |
| Global CLI            | App repositories outside this monorepo | `npm i -g @lattice-ui/cli` | `lattice <command>`      |
| Monorepo Local Script | Working inside this repository         | No global install required | `pnpm lattice <command>` |

If you do not want a global install, you can also run one-off commands with `npx lattice <command>`.

## Verify CLI Access

### Global CLI

```bash
npm i -g @lattice-ui/cli
lattice --help
```

### Monorepo Local Script

```bash
pnpm lattice --help
```

## Project Creation

### Global CLI

```bash
lattice create
```

### Monorepo Local Script

```bash
pnpm lattice create
```

`create` is interactive by default.
Use optional flags like `--pm`, `--git`, `--lint`, and `--no-lint` only when you need explicit non-default behavior.

After `create`, generated projects include `@lattice-ui/cli` in `devDependencies`, so local command execution is immediately available:

```bash
npx lattice --help
pnpm lattice --help
yarn lattice --help
```

## Install Component Packages

### Global CLI

```bash
lattice add core,style,system
lattice add dialog,toast --preset overlay
```

### Monorepo Local Script

```bash
pnpm lattice add core,style,system
pnpm lattice add dialog,toast --preset overlay
```

## Remove Component Packages

### Global CLI

```bash
lattice remove dialog
lattice remove --preset overlay --dry-run
```

### Monorepo Local Script

```bash
pnpm lattice remove dialog
pnpm lattice remove --preset overlay --dry-run
```

## Validate and Upgrade

### Global CLI

```bash
lattice doctor
lattice upgrade --preset form
```

### Monorepo Local Script

```bash
pnpm lattice doctor
pnpm lattice upgrade --preset form
```

## Peer Dependencies

Ensure compatible peers are installed:

```bash
pnpm add @rbxts/react @rbxts/react-roblox
```
