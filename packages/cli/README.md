# @lattice-ui/cli

Node CLI for Lattice UI projects.

## Install

```bash
npm i -g @lattice-ui/cli
```

## Usage

```bash
lattice <command> [options]
```

### Commands

- `lattice init`
- `lattice add <name...> [--preset <preset...>]`
- `lattice upgrade [name...] [--preset <preset...>]`
- `lattice doctor`

### Global options

- `--cwd <path>`
- `--pm <pnpm|npm|yarn>`
- `--verbose`
- `--dry-run`
- `--yes`
