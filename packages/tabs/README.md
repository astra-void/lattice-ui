# @lattice-ui/tabs

Behavior-only Tabs primitives for Roblox UI, aligned with the lattice compound pattern.

## Exports

- `Tabs`
- `Tabs.Root`
- `Tabs.List`
- `Tabs.Trigger`
- `Tabs.Content`

## Notes

- `orientation` supports `horizontal` and `vertical` trigger navigation.
- Enabled triggers are selectable and activate immediately when selection focus moves onto them.
